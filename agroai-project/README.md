# 🌿 AgroAI — Precision Farming Intelligence

A full-stack React web application for AI-powered precision agriculture.

## ✅ Quick Start

### Prerequisites
- Node.js (v16 or higher): https://nodejs.org
- npm (comes with Node.js)

### Run the App

```bash
# 1. Install dependencies
npm install

# 2. Start the development server
npm start
```

The app will open at **http://localhost:3000**

## 🔑 Features
- 🔐 Login & Registration
- 🏠 Home Page with animated hero
- ℹ️ About Page with team profiles
- ⚙️ Features Page with drag & drop AI crop prediction
- 📊 Dashboard with live charts & satellite field map
- 🌦️ Weather Intelligence with 7-day forecast
- 📞 Contact Page with interactive form
- 🤖 AI Chatbot (floating)
- 🎙️ Voice Assistant
- 🌍 5 Language Support (EN, Hindi, Marathi, Gujarati, Tamil)
